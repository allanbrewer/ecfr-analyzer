About the eCFR Analyzer

Department of Government Efficiency

Making government work better

Project Overview

The eCFR Analyzer is a tool developed by the Department of Government Efficiency (DOGE) to analyze the Electronic Code of Federal Regulations. This project aims to identify opportunities for regulatory streamlining by examining the language, complexity, and historical changes in federal regulations.

Using data directly from the eCFR API and the govinfo.gov bulk data API, we provide insights into regulatory trends, agency activities, and language patterns that may indicate inefficiencies or opportunities for improvement.

Key Features

  • Comprehensive word count analysis by agency
  • Historical tracking of regulatory changes over time
  • Analysis of corrections and amendments
  • DEI footprint analysis
  • Waste and inefficiency identification
  • Bureaucratic complexity metrics
  • Advanced search capabilities
Data Sources
eCFR API

The Electronic Code of Federal Regulations API provides access to the most current version of the CFR. This API allows us to retrieve the full text of regulations, their structure, and metadata.

View eCFR API Documentation
GovInfo Bulk Data

The govinfo.gov bulk data repository provides access to historical versions of the CFR, enabling analysis of how regulations have changed over time.

View GovInfo Bulk Data
GovInfo API

The GovInfo API provides programmatic access to metadata and content published on the govinfo.gov website.

View GovInfo API Documentation
Methodology
Data Collection

The eCFR Analyzer downloads data from official government APIs, processes it, and stores it locally for analysis. Historical data is retrieved from the GovInfo bulk data repository to enable time-series analysis.

Analysis Techniques

Our analysis includes:

  • Word Count Analysis: Quantifies the volume of regulations by agency.
  • Keyword Analysis: Identifies specific terms related to DEI, waste, and bureaucracy.
  • Complexity Metrics: Measures sentence length, vague language usage, and other readability factors.
  • Historical Comparison: Tracks changes in regulations over time to identify trends.
Data Updates

The website is automatically updated on a daily basis using GitHub Actions, ensuring that the analysis reflects the most current regulatory environment.

Limitations and Disclaimers
Data Limitations

While we strive for accuracy, the analysis has certain limitations:

  • The analysis is based on text patterns and may not capture the full context of regulations.
  • Historical data availability varies by agency and title.
  • API rate limits may affect the comprehensiveness of some analyses.
  • Keyword analysis is based on predetermined sets of terms that may not capture all relevant language.
Disclaimer

This tool is intended for informational purposes only. While the data is sourced from official government APIs, the analysis and interpretations are those of the Department of Government Efficiency and should not be considered official government positions. Always refer to the official eCFR website for the authoritative version of any regulation.
